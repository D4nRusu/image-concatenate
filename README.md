# image-concatenate

## how to use

1. Make sure you have Python 3 installed.

2. Create a virtual environment:
   python -m venv venv

3. Activate it:
   - Windows: venv\Scripts\activate
   - macOS/Linux: source venv/bin/activate

4. Install the requirements:
   pip install -r requirements.txt

5. Run the script:
   python script.py
